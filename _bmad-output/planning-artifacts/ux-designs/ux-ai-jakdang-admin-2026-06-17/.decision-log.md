### 사용자 결정(확정 2026-06-17)
1. **권한 경계** = 운영자(staff)는 **숨김(soft)까지**, **영구삭제·사이트설정·광고·회원 권한변경 = 최고관리자 전용**. 권한 없는 메뉴/액션은 숨김 + 접근 시 거부. (3단계 모델 유지하되 운영자 상한선 = 숨김.)
2. **파괴적 확인 UX** = **위험도별 차등**. 되돌릴 수 있는 숨김·플래그 토글 = 즉시 실행 + 토스트(undo 노출). 삭제·이용제한·포인트차감·배지회수 = **확인 모달 + 사유 메모 입력**. 신고처리 5단계 가드레일과 일관.
3. **삭제 보존** = **soft-delete 기본 + 보존기간(예: 30일) 후 자동 영구삭제(hard-delete)**. 보존 기간 내 복구(restore) 가능, "휴지통 자동 비움". 보존기간 값은 사이트 설정(신고/콘텐츠 설정) 노출 후보 `[NOTE FOR UX]`.
