운영정책 제3조 (금지 행위) — 다음 콘텐츠·행위를 명시적으로 금지합니다.

  • 콘텐츠: 음란·선정적 콘텐츠, 폭력적·혐오적·차별적 내용, 개인정보 무단 게시, 저작권 무단 복제, 허위 정보·명예훼손.

운영정책 제5조 (신고 처리 절차) — 신고 접수 → 운영팀 1차 검토(영업일 3일 이내) → 처리 결정 → 신고자 통보. 허위 신고자도 제재.

운영정책 제6조 (제재 정책) — 위반 시 단계적 제재.

  • 경고 + 포인트 차감 → 경고 누적 3회 시 7일 정지, 5회 시 30일 정지.
  • 중대 위반(불법 콘텐츠·개인정보 침해·반복 중대 위반)은 즉시 영구 정지.
  • 정지 회원은 로그인·게시·댓글·쪽지가 제한됩니다.
  • 이의 신청 절차(14일 이내 1:1 문의) 제공.

3-2. 실제 구현된 콘텐츠 관리 시스템

운영정책은 선언에 그치지 않고, 아래 기능이 실제 시스템으로 구현·작동하고 있습니다.

  • 금칙어 자동 마스킹 — 게시글·댓글 작성 시 금칙어(욕설·비속어)를 자동 감지하여 *로 가림(네이버 댓글과 동일 방식).
  • 스팸·도배 링크 자동 차단 — 과도한 URL 또는 단축 URL 포함 시 등록 차단.
  • 회원 신고 기능 — 모든 게시글·댓글·질문·답변·자료·쪽지·회원 프로필에 신고 버튼 제공. 누적 신고 임계치 초과 시 대상 콘텐츠 자동 숨김.
  • 누적 제재 시스템 — 위반 누적 시 경고 자동 발부, 관리자 판단으로 정지·영구정지 단계적 제재.
  • 관리자 검수·숨김·삭제 — 운영자 전용 관리 페이지에서 신고 콘텐츠 검토·숨김·삭제·숨김 해제 처리.
  • AI 생성 콘텐츠 사전 검열 — 초기 시딩 AI 생성 글은 검열 파이프라인(부적절 시 보류·폐기)을 거쳐야 게시.
위 시스템을 통해 음란물·스팸·혐오·불법 콘텐츠 등이 서비스에 노출되지 않도록 사전 필터링 + 사후 신고·제재의 이중 안전장치를 운영하고 있습니다.

4. 네이버 로그인으로 수집하는 정보 및 이용 목적

  • 이용 목적: 회원 식별·가입, 커뮤니티 활동(게시·댓글·신고 처리), 본인 확인 및 서비스 공지.
