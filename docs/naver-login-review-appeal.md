## 2. 게시물 성격(유형) 및 카테고리별 게시 콘텐츠

서비스는 6개 대메뉴 아래 총 12개 게시판으로 구성되며, 각 게시판에 게시되는 콘텐츠는 다음과 같습니다.

### ① 바이브 코딩
- **바이브 코딩 가이드 / 팁** — AI에게 코딩을 맡겨 개발하는 실무 방법론, 튜토리얼, 노하우 텍스트 게시글.

### ② AI 자동화
- **자동화 가이드 / 사례 / 팁** — AI 업무 자동화 워크플로 구축 방법, 실제 적용 사례, 도구 추천 게시글.

### ③ AI 수익화
- **수익화 팁 / 사례** — AI를 활용한 수익 창출 아이디어와 실제 경험담 공유 게시글.
  (합법적 부업·외주·콘텐츠 판매 등 정보 공유이며, 투자 권유·금융 상품과 무관합니다.)

### ④ 묻고답하기 (Q&A) — `/questions`
- AI 활용 중 발생한 오류 해결, 방향성 검토, 수익화 상담 등 **질문/답변**. 답변 채택 기능 제공.

  다운로드형 자료실. 평점·후기로 자료 품질을 검증.

### ⑥ 작당 라운지
- **공지사항** — 운영자만 작성하는 시스템 공지.
- **AI 창작마당** — AI로 생성한 이미지·영상·음악 등 창작물 공유.

## 3. 부적절 콘텐츠 방지 운영정책 및 관리 시스템 (소명)

AI작당은 SNS·포털 서비스의 정상 운영을 저해하거나 부적절한 콘텐츠가 등록되지 않도록
**공개된 운영정책**과 **다층 콘텐츠 관리 시스템**을 실제로 운영하고 있습니다.

### 3-1. 공개 운영정책 · 약관 (비회원도 열람 가능)

다음 3종 문서를 웹사이트에 상시 게시하고 있으며, 로그인하지 않아도 누구나 열람할 수 있습니다.

| 문서 | URL |
|---|---|
| 이용약관 | https://aijakdang.com/terms |
| 개인정보처리방침 | https://aijakdang.com/privacy |
| **커뮤니티 운영정책** | https://aijakdang.com/operation-policy |

**운영정책 제3조 (금지 행위)** — 다음 콘텐츠·행위를 명시적으로 금지합니다.
- 콘텐츠: 음란·선정적 콘텐츠, 폭력적·혐오적·차별적 내용, 개인정보 무단 게시,
  저작권 무단 복제, 허위 정보·명예훼손.
- 활동: 스팸 반복 게시, 다중 계정 악용, 포인트 부정 취득, 광고 도배,
  비정상 자동화 접근(크롤링), 스팸 쪽지, 불법 소프트웨어 배포 링크.

**운영정책 제5조 (신고 처리 절차)** — 신고 접수 → 운영팀 1차 검토(영업일 3일 이내)
→ 처리 결정 → 신고자 통보. 허위 신고자도 제재.

**운영정책 제6조 (제재 정책)** — 위반 시 단계적 제재.
- 경고 + 포인트 차감 → 경고 누적 3회 시 7일 정지, 5회 시 30일 정지.
- 중대 위반(불법 콘텐츠·개인정보 침해·반복 중대 위반)은 즉시 영구 정지.
- 정지 회원은 로그인·게시·댓글·쪽지가 제한됩니다.
- 이의 신청 절차(14일 이내 1:1 문의)도 함께 제공합니다.

### 3-2. 실제 구현된 콘텐츠 관리 시스템

운영정책은 선언에 그치지 않고, 아래 기능이 **실제 시스템으로 구현·작동**하고 있습니다.

1. **금칙어 자동 마스킹** — 게시글·댓글 작성 시 금칙어(욕설·비속어 등)를 자동 감지하여
   `*`로 가립니다. (네이버 댓글과 동일한 방식으로 관리자 설정 목록 기반 필터링)

2. **스팸·도배 링크 자동 차단** — 한 게시물에 과도한 URL 또는 단축 URL이 포함되면
   등록을 차단하여 스팸·광고 도배를 방지합니다.

3. **회원 신고 기능** — 모든 게시글·댓글·질문·답변·자료·쪽지·회원 프로필에 신고 버튼을 제공합니다.
   신고 사유(스팸/욕설·비방/개인정보 노출/허위정보/사칭 등)를 선택해 접수하며,
   **누적 신고가 임계치를 넘으면 대상 콘텐츠가 자동으로 숨김 처리**됩니다.

4. **누적 제재 시스템** — 위반 신고가 누적되면 경고가 자동 발부되고,
   관리자 판단에 따라 정지·영구정지로 단계적으로 제재합니다. (정지 회원은 서비스 이용 제한)

5. **관리자 검수·숨김·삭제** — 운영자 전용 관리 페이지에서 신고 콘텐츠를 검토하여
   숨김·삭제·숨김 해제를 처리할 수 있습니다.

6. **AI 생성 콘텐츠 사전 검열** — 서비스 초기 콘텐츠 시딩에 사용되는 AI 생성 글은
   별도 검열 파이프라인(부적절 판정 시 보류·폐기)을 거쳐야만 게시됩니다.

> 위 시스템을 통해 음란물·스팸·혐오·불법 콘텐츠 등이 서비스에 노출되지 않도록
> 사전 필터링 + 사후 신고·제재의 이중 안전장치를 운영하고 있습니다.

- 미동의(미제공) 항목은 저장하지 않으며 기존 값을 덮어쓰지 않습니다.
